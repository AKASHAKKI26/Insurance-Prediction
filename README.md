# Insurance Prediction

## Project Overview

Insurance Prediction is a machine learning project that analyzes historical data to predict insurance costs. The project uses data preprocessing, feature engineering, and regression algorithms to estimate future insurance charges.

A simple web interface is built using **Streamlit** where users can enter details and get predicted insurance cost instantly.

---

## Features

* Data preprocessing and cleaning
* Feature engineering
* Machine learning model training
* Insurance cost prediction
* Interactive Streamlit web application

## Installation

Clone the repository:

```
git clone https://github.com/AKASHAKKI26/Insurance-Prediction.git
```

Navigate to the project folder:

```
cd Insurance-Prediction
```

Install required packages:

```
pip install -r requirements.txt
```

---

## Run the Application

Start the Streamlit app using:

```
streamlit run app.py
```

After running the command, the application will open in your browser.

---

## How the Application Works

1. The user enters input values in the Streamlit interface.
2. The machine learning model processes the input data.
3. The model predicts the insurance cost.
4. The result is displayed instantly on the web interface.
